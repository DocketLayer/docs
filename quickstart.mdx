## Prerequisites

- A Solana wallet funded with USDC — see the [wallet setup guide](/guides/wallet-setup)
- A case identifier and court code for a covered court (US state, federal, or Canadian)

---

## Step 1 — Check coverage

Before querying, confirm the court you need is covered. This endpoint is free and requires no payment.

```bash
curl https://api.docketlayer.ai/v2/status
```

Find your `court_code` in the response. If your court shows `"coverage": "full"` you are ready. If your court is not listed, check the [coverage roadmap](/coverage-roadmap).

---

## Step 2 — Try sandbox mode

Before spending USDC, confirm connectivity with a free sandbox query. Add `?test=1` to any endpoint.

```bash
curl "https://api.docketlayer.ai/v2/case?test=1&case_id=1:24-cv-01234&court_code=nysd"
```

The response is a fixture — real structure, no real data, no charge.

---

## Step 3 — Make your first paid query

DocketLayer uses the [x402 payment protocol](/x402) — payment is a signed Solana USDC transaction attached to the request. The flow is: send request → receive 402 with payment requirements → sign and attach payment → retry.

The easiest path for AI coding environments is the [MCP npm package](/guides/mcp-server), which handles payment automatically. For direct REST integration:

<CodeGroup>

```typescript TypeScript
import bs58 from "bs58";
import { createKeyPairSignerFromBytes } from "@solana/signers";
import { ExactSvmScheme } from "@x402/svm";
import { x402Client, x402HTTPClient } from "@x402/core/client";

// Set up the x402 payment client once
const keyBytes = bs58.decode(process.env.DOCKETLAYER_WALLET_PRIVATE_KEY!);
const signer = await createKeyPairSignerFromBytes(keyBytes);

const scheme = new ExactSvmScheme(signer);
const client = new x402Client();
(client as any).register("exact", scheme);
const httpClient = new x402HTTPClient(client);

// Make the initial request — expect 402
const url =
  "https://api.docketlayer.ai/v2/case" +
  "?case_id=1:24-cv-01234&court_code=nysd&last_checked=2026-01-01T00:00:00Z";

const initial = await fetch(url);
if (initial.status !== 402) throw new Error(`Unexpected status ${initial.status}`);

// Parse requirements, sign, and retry
const requirements = JSON.parse(initial.headers.get("X-Payment-Requirements")!);
const payload = await httpClient.createPaymentPayload(requirements);
const paymentHeaders = httpClient.encodePaymentSignatureHeader(payload);

const response = await fetch(url, { headers: paymentHeaders });
const data = await response.json();

if (data.delta?.changed) {
  for (const filing of data.delta.new_filings) {
    console.log(`${filing.filing_type}: ${filing.description}`);
  }
} else {
  console.log("No changes");
}
```

```bash curl (sandbox)
curl "https://api.docketlayer.ai/v2/case?test=1&case_id=1:24-cv-01234&court_code=nysd&last_checked=2026-01-01T00:00:00Z"
# For live paid queries, use the TypeScript x402 libraries or the MCP npm package
```

</CodeGroup>

**Dependencies for the TypeScript example:**

```bash
npm install bs58 @solana/signers @x402/svm @x402/core
```

## Step 5 — Handle errors

DocketLayer never charges for failed queries.

```typescript
if (response.status === 402) {
  // Payment failed — check wallet balance and x402 configuration
} else if (response.status === 404) {
  // Case not found — verify case_id format and court_code
} else if (response.status === 422) {
  // Court not covered — check /v2/status for current coverage
} else if (response.status === 429) {
  // Rate limited — back off, retry after retry_after interval
} else if (response.status === 503) {
  // Upstream court system unavailable — retry later, check /v2/status
}
```
